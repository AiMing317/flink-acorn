### 快速使用

?> 参考案例项目: `https://github.com/ispong/flink-acorn/tree/main/examples/acorn-demo`

##### 添加依赖

```xml
<dependency>
    <groupId>com.isxcode.acorn</groupId>
    <artifactId>acorn-client</artifactId>
    <version>1.2.0</version>
</dependency>
```

###### 配置服务信息

```yaml
acorn:
  check-servers: true
  servers:
    default:
      host: isxcode
      port: 30155
      key: acorn-key
```

##### 插件使用

```java
public class demo {
    
}
```
