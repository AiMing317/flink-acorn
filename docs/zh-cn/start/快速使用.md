### 快速使用

#### 前提


[安装hadoop]()

[安装flink]()

##### 服务器下载源码

```bash
git clone xxx
./build.sh

ln -s /opt/
```

###### 配置环境变量

```bash
export FLINK_HOME=/usr/local/flink
```

###### 配置插件

```bash
acorn config
```

```yaml
server:
  port: 30155

acorn:
  key: acorn-key 
```

###### 启动插件

```bash
acorn start
```

#### 添加依赖

```xml
<dependency>
    <groupId>com.isxcode.acorn</groupId>
    <artifactId>acorn-client</artifactId>
    <version>12.0</version>
</dependency>
```

```yaml
acorn:
  check-servers: true
  servers:
    default:
      host: 192.168.66.66
      port: 30155
      key: acorn-key
```
