离线部署

###### 前提

- 安装java
- 安装maven
- 安装unzip

###### 下载flink安装包
 
```bash
unzip flink-package.zip
安装flink flink-package/flink-1.14.0-bin-scala_2.12.tgz
```

###### 配置maven

```bash
mvn --version
cp ~/flink-package/settings.xml /opt/maven/conf/settings.xml
```

###### 复制repository下面的包

```bash
unzip ~/flink-package/repository.zip -d ~/.m2/
```

##### 启动插件

```bash
bash deploy.sh
```
