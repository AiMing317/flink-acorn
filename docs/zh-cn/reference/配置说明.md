## 插件配置

```yml
server:
  port: 30155 # 插件端口号

acorn:
  plugin:
    key: acorn-key # 插件密钥
    flink-host: 127.0.0.1 # flink的host
    flink-port: 8081 # flink的端口号
    tmp-dir: /tmp/acorn # 动态生成的项目临时存储的位置
    hive-conf-path: /opt/flink/conf # hive-site.xml的存储位置
    storage-tmp: false # 是否保存临时项目文件

logging:
  group:
    acorn: com.isxcode.acorn
  level:
    acorn: debug
```

## 客户端配置

```yml
acorn:
  worker:
    check-server: true # 是否检查插件状态
    workers:
      default: # 作业的名字
        host: 192.168.66.66 # 插件的ip
        port: 30155 # 插件的端口号
        key: acorn-key # 插件的密钥
      worker1:
        host: 192.168.66.66
        port: 30155
        key: acorn-key  
```
