#### 如何加入我们

1. fork一下项目。
2. 创建一个合并请求（Pull Request），修改`.mailmap`文件，类似下面的格式。

```text
ispong <ispong@outlook.com>
```

#### 开启本地文档

```bash
# 先安装nodejs
npm i docsify-cli -g
docsify serve docs
```

#### 构建开发环境

参考小栗子博客，安装flink，kafka，mysql等测试项目。
